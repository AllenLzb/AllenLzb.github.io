一、Java 基础

1.JDK 和 JRE 有什么区别？
JDK面向开发人员,kre面向使用者
编写的java程序必须要jre才 能与运行
安装位置不同

2.== 和 equals 的区别是什么？
等等基本的类型比较的是值是否相等,引用类型比较地址值是否相等
equals重写前比较的地址值,重写后比较的是对象的属性

3.两个对象的 hashCode()相同，则 equals()也一定为 true，对吗？
不对
hashCode相等equal不一定相等因为hashcode并不完全可靠
equals相等HashCode也相等因为equals返回对象是否相等,hashcode是返回对象的哈希值,

4.final 在 java 中有什么作用？
用于修饰类,方法,变量.并且不可修改
修饰类 该类不能被继承
修饰方法 方法不能被重写
修饰变量 变量不能被修改

5.java 中的 Math.round(-1.5) 等于多少？
-1
round是四舍五入
大于0.5 舍去小数 绝对值+1
小于0.5 舍去小数
等于0.5 加0.5

6.String 属于基础的数据类型吗？
不属于
String属于引用类型
8基本数据类型 byte short int long float double boolean char

7.java 中操作字符串都有哪些类？它们之间有什么区别？
String、StringBuffer、StringBuilder
我先说一下相同点吧 buffer 和 builder
都是字符串缓冲区,都可变的字符序列,都具有相同的构造和方法
不同点就是他们的内存有差别
String是不可变对象,每次操作都会生成新的String对象,然后指针指向新的String对象
buffer和builder 可以在原有对象上进行操作,所以在经常改变字符串内容的场景下最好不要使用String
Buffer是JDK1.1出现
Builder是JKD1.5出现
Buffer线程安全,有同步锁,多线程仍然可以保证数据安全
Builder线程不安全,多线程无法保证数据安全
Builder效率大于Buffer String效率最低
比较多的就是用Buffer 进行添加,删除,替换和反转

8.String str=”i”与 String str=new String(“i”)一样吗？
不一样
内存分配方式不一样
String s = ‘i’方式 Java虚拟机将其分配到常量池当中
newString方式是分配到堆内存当中

11.抽象类必须要有抽象方法吗？
不需要
抽象类不一定有抽象方法,包含抽象方法的类一定是抽象类

12.普通类和抽象类有哪些区别？
1.抽象类不能被实例化
2.抽象类的抽象方法无需实现
3.包含抽象方法一定是抽象类
4.如果没有实现抽象父类所有抽象方法那么子类就是抽象类.
5.抽象方法不能声明static
6.抽象方法不能用private和final修饰

13.抽象类能使用 final 修饰吗？
不能
抽象类用于继承的,final修饰的类无法继承

14.接口和抽象类有什么区别？
1.抽象类需要被子类继承,接口需要被子类实现
2.接口只能做方法声明,抽象类中可以做方法声明,也可以做方法实现
3.接口的变量只能是公共的静态常量,抽象类中的变量是普通变量
4.接口是设计的结果,抽象类是重构的结果
5.接口的抽象级别最高
6.抽象类可以具有方法和属性,接口只能有抽象方法和不可变常量
7.抽象类主要用来抽象类别,接口主要用来抽象功能

15.java 中 IO 流分为几种？
四种
字节输入流InputStream
字节输出流OutputStream
字符输入流Reader
字符输出流Writer

16.BIO、NIO、AIO 有什么区别？
这是java支持的三种网络编辑模式
Bio同步并阻塞 一个连接对应一个线程
Nio同步非阻塞 一个线程处理多个请求
Aio异步非阻塞 适用于连接数较多且连接时间较长

exits 检测文件路径是否存在
createFile 创建文件

delete 删除一个文件或目录

二、容器

18.java 容器都有哪些？
collection(kelaikeshen)和map

19.Collection 和 Collections 有什么区别？
collection是单列集合的顶级接口 派生set和list 具有单列集合中共有的功能
collections是一个集合类的工具类,提供很多静态方法

20.List、Set、Map 之间的区别是什么？
list 允许重复,有序,有索引
set 不允许重复,无序,无索引
map是双列集合的顶级接口 储存键值对 键唯一 值不唯一

21.HashMap 和 Hashtable 有什么区别？
都是储存键值对,都是map实现类,都使用哈希表的方式储存

线程安全性不同,hashtable是线程安全的,每个方法中都加入了synchronize方法
hashmap不是线程安全的,在多线程并发环境下,可以产生死锁

hashmap效率高

hashtable继承dictionary hashmap继承AbstractMap

hashtable可以使用Enumeration遍历

hashtable初始值是11 每次扩容2n+1 hashmap初始值是16 每次扩容2n

22.如何决定使用 HashMap 还是 TreeMap？
需要得到一个有序的结果 使用treemap
不需要排序就是用hashmap

23.说一下 HashMap 的实现原理？
hashmap是以hash表结构储存
可以存null
底层hashcode方法 结构是数组加链表红黑树

24.说一下 HashSet 的实现原理？
父类接口是set集合
以hash表结构储存
多线程不安全
可以存null
底层使用hashmap实现

25.ArrayList 和 LinkedList 的区别是什么？
相同点他们都是单列集合list接口的实现类,都是有序的,有索引的,可重复的

数据结构不同 arraylist数据结构是动态数组 listlist数据结构是双向链表

随机访问方式不同 linklist是线性的存储方式,需要指针从前往后依次查找
所以arraylist比linklist随机访问的时候效率要高

增加和删除时arraylist影响其他数组下标,所以linklist要比arraylist效率要高

一般情况下 在需要频繁读取集合中元素时使用arraylist
再插入和删除操作时使用linkedlist

27.ArrayList 和 Vector 的区别是什么？
vector是线程安全的 有synchronized 而arraylist是线程不安全的
vector扩容是原来的一倍 arraylist是0.5倍

一般情况下
如果只有一个线程访问到集合 使用arraylist 如果有多个线程访问 使用vector
这样不需要考虑和编写线程安全的代码

28.Array 和 ArrayList 有何区别？

arry是数组 arraylist是集合
array包含基本对象和对象类型 arraylist只能包含对象类型
arraylist集合可以自动扩容 array数组创建后不可更改
arraylist提供很多方法 比如 addall removeall iterator
array类型的变量在声明同时必须进行初始化 arraylist可以先声明
arraylist提供自动装箱

三、多线程

35.并行和并发有什么区别？
并行是多cpu时,一个cpu执行一个进程,而另一个cpu可以执行另一个进程.两个进程不抢占cpu资源.可以同时进行
并发时多个任务在同一个cpu上,按时间切片轮流交替执行

他们的区别就是
并行 多个事情 在同一时间点上同时发生,且多个任务之前不互相抢占资源
并发 多个事情 在同一时间点上同时发生 , 多个任务之前互相抢占资源

36.线程和进程的区别？

线程是程序执行的基本单元

37.守护线程是什么？

38.创建线程有哪几种方式？

实现runnable接口 重写run方法
实现callable接口 通过包装器来创建thread线程

39.说一下 runnable 和 callable 有什么区别？

40.线程有哪些状态？

41.sleep() 和 wait() 有什么区别？
都可以让线程暂停执行
同步锁不同
sleep后程序不释放同步锁
wait后会释放同步锁
用法不同
sleep可以用时间指定来自动醒过来
wait可以用notify唤醒
wait的notify,notifyall只能在同步控制方法或者同步代码块中使用sleep可以在任何地方使用
不同类属
sleep类是thread
wait类是object
